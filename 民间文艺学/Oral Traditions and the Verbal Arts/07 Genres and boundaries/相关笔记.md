# 第七章核心术语中英对照及界定清单

|英文术语|中文译法|核心界定（基于原文核心内容）|
|---|---|---|
|Genre|文类|口头或书面表达的约定俗成类别，基于风格、主题、场合、表演特征等标准划分，具有相对稳定性但非绝对固定，是沟通中可调用的资源（如神话、史诗、民间故事等）。|
|Myth|神话|散文叙事，在讲述社会中被视为对遥远过去事件的真实记述，具有神圣性，核心角色多为非人类（神祇、动物、文化英雄），与神学、仪式相关，用于解释起源（世界、人类、自然现象）或作为信仰权威。|
|Legend|传说|散文叙事，被视为对近期过去真实事件的记述，核心角色为人类，空间维度是当下世界，态度可世俗可神圣，近年也涵盖当代都市传说。|
|Folktale|民间故事|散文叙事，被视为虚构作品，不被当作教义或历史，可设定于任何时间地点，角色可为人类或动物，无严格受众限制（非仅面向儿童），包含奇幻元素但不等同于 “童话”。|
|Epic|史诗|源于古希腊文学，指长篇叙事诗（或相关诗歌集群、诗歌传统），以英雄主题为核心，分口头（原生）和书面（次生）形式，长度、诗体属性等界定存在争议，部分文化中与民族身份紧密相关。|
|Riddle|谜语|简短口头文类，以问答或隐喻形式呈现，具有智力挑战性，核心是隐藏真实指代，需通过联想或逻辑推理破解，是言语文类的重要分支。|
|Proverb|谚语|凝练的口头表达，承载群体经验或道德准则，形式简洁、易于记忆，在日常交流中可作为劝诫、论证的依据，近年研究更关注其使用语境与解读。|
|Narrative|叙事|涵盖所有包含时间序列的表达形式（口头 / 书面、虚构 / 非虚构），不仅是特定文类，更是贯穿多种媒介的跨文类主题，核心是通过情节、角色传递意义。|
|Panegyric/Praise Poetry|颂诗 / 赞美诗|以赞颂为核心功能的诗歌文类，多在特定场合（聚会、仪式）表演，面向特定对象（人物、群体、神祇），强调对象的功绩、美德或价值。|
|Lament|挽歌|用于哀悼场景（葬礼、丧亲）的文类，常伴随特定情感表达（呜咽、悲叹），形式可诗可歌，核心是抒发悲伤情绪，部分文化中具有仪式功能。|
|Cross-generic Themes|跨文类主题|不局限于单一文类的核心线索，贯穿多种表达形式，如 “叙事”“神话性想象”“戏剧性表演” 等，是表面文类差异之下的共性元素。|
|Oral/Written|口头 / 书面|区分表达媒介的核心界限，口头形式强调即时性、表演性、互动性，书面形式侧重固定性、持久性；二者非绝对二分，而是相互影响的连续体。|
|Prose/Poetry|散文 / 诗歌|基于语言形式的界限划分：散文无明显韵律、诗行结构，诗歌多具节奏、韵律、诗节等特征；口头语境中需结合表演、韵律、本土评价判断，而非仅依赖书面排版。|
|Speech/Song|言说 / 歌唱|口头表达的两种核心模式，前者侧重自然语言节奏，后者含旋律、曲调元素，二者间存在过渡形式（吟诵、吟唱），部分文类结合或交替使用两种模式。|
|Ballad|民谣|短篇叙事诗歌，多为口头流传，主题涵盖爱情、英雄事迹、日常事件等，形式简洁，常伴随简单韵律，部分文化中与民间生活紧密相关。|
|Narrative Prose|叙事散文|以叙事为核心功能的散文形式，包含情节、角色、时间序列，如民间故事、传说中的核心表达载体，区别于诗歌化的叙事表达。|
|Cross-generic Analysis|跨文类分析|不孤立研究单一文类，而是将社群内所有文类（含视觉、物质形式）置于同一框架比较，揭示本土范畴间的关系及跨文类共性。|
