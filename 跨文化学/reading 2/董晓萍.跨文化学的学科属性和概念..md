### （一）研究背景与核心议题

1. **民俗学的双重属性**：民俗学本是国际性学科，历史上侧重本土研究 —— 既成就其学科根基（如服务国家建设、民族文化传承），又因本土化局限产生三大未决问题，成为学科发展瓶颈：
    - 未解决 “分布式民俗类型与异文式差异文本” 的传承矛盾，核心理论 “世界分布式类型学说” 存在漏洞；
    - 未构建 “民俗史、家庭史与社会史关系” 的理论模型，在人文学科群中 “失语”；
    - 习惯同质民俗研究，难以应对现代社会 “农业同质民俗与现代异质民俗混合” 的现状，非遗保护多为应用研究，缺乏基础理论突破。
2. **跨文化学的价值**：可解决民俗学长期无法突破的瓶颈，同时民俗学的国际化经验也能为跨文化学建设提供支撑，二者结合可实现学科互哺。

### （二）民俗学的发展历程与局限

1. **国际民俗学流派演进**：各流派虽推动民俗学国际化，但均未突破 “非跨文化” 局限：
    
    |流派|代表成果|核心局限|
    |---|---|---|
    |英国民俗学|首创 “民俗（Folklore）” 概念、成立首个民俗学会、编纂首部《民俗学手册》|未及时构建自身理论体系，吸引力弱于人类学|
    |德国民俗学|依托浪漫主义文学、维柯历史哲学、格林兄弟工作，将语言学 / 哲学引入民俗学|未形成突破地域局限的研究框架|
    |芬兰学派（世界民俗学代表）|设立首个民俗学教席、提出 “故事类型学”、建立国际民俗学会|①欧洲一元论（认为故事起源于欧洲，强制非欧洲国家遵循欧洲模式）；②文学化倾向（聚焦文学文本，排斥文化 / 社会研究）|
    |美国 / 法国 / 俄罗斯民俗学|美国帕里 - 洛德学派《故事的歌手》、法国列维 - 斯特劳斯《野性的思维》、俄罗斯普罗普 “故事形态学”|仍以本国 / 区域研究为主，未实现 “跨文化” 整合|
    |中国民俗学|钟敬文创立中国民俗学派，以国学为基础，融合中外学说，研究口头传统与历史文献|早期受外来学说影响，虽立足本土，但未充分拓展 “跨国”“多极点” 研究视角|
    
2. **民俗学未发展为跨文化学的关键原因**：20 世纪受 “一战”“二战” 影响，民俗学承担 “本土国家解放、民族独立” 使命，形成 “国家学”（辅助国家建设）与 “文化学”（共建多民族文化）双重定位，长期侧重本土研究，难以突破地域边界。

### （三）跨文化学的学科属性

跨文化学吸收民俗学国际化优势，突破其本土局限，形成 11 项核心学科属性转变：

|维度|民俗学传统属性|跨文化学新属性|
|---|---|---|
|研究议题|本土问题|世界多边交流问题|
|价值立场|本土立场|人类社会共同发展立场|
|研究范围|一国研究|人类共同体研究|
|理论基础|工具论|对话论|
|研究对象|纯 “原生态” 同质民俗|同质与异质混合的可改写 / 可编辑民俗|
|研究视角|传统权威视角|宏观统一视角下的标准研究|
|研究群体|民众属地人口|异地网民群|
|研究场景|农村|世界城市|
|符号研究|无限异式分布式|无疆界有限文化符号|
|研究性质|非遗应用研究|历史性研究|
|研究层次|农业 / 工业 / 技术民俗研究|价值研究（升华文化内核）|

_注_：跨文化学≠复兴芬兰学派，需先解决芬兰学派遗留的 “欧洲中心论”“文学化倾向” 等瓶颈，才能形成新理论支撑。

### （四）跨文化学的基本概念（与民俗学概念重组）

跨文化学通过 “概念重组” 激活民俗学国际化优势，形成三大核心概念体系：

1. **“跨国” 与 “类型” 重组**
    
    - “跨国” 的双重内涵：①超越国家 / 种族边界，遴选人类共享文化（如故事类型），推动人文精神升级；②摒弃欧洲一元论，建立非欧中心的普适性理论，促进文化互鉴。
    - 典型案例：疫情中的 “口罩故事”—— 中西口罩观差异催生多元民谣、表演，后通过全球防疫合作实现 “故事类型跨文化”，成为人类命运共同体建设的文化载体。
    - 衍生概念群：跨文化论、自我工具论、文化丛论、文化自觉论、二元结构论、文化含义网论。
2. **“多极点” 与 “一国” 重组**
    
    - “多极点” 的核心作用：衔接 “民俗学文化多样性研究” 与 “一国民俗研究”，突破 “人类共有民俗研究禁区”（如婚制、继承制、丧葬制的世界共性）。
    - 突破方向：①纠正英国民俗学 “将民俗视为原始文化化石” 的静态认知；②填补 “民俗史 - 家庭史 - 社会史” 关系研究空白（参考马克思、恩格斯人类社会史理论，结合故事类型中的家庭民俗素材）；③区分 “多元来源民俗”（如经历分裂 / 侵略的国家）与 “连续历史来源民俗”（如中国），针对性开展文化圈研究。
    - 衍生概念群：多义情境、自我信仰、动态知识论。
3. **“文化转场” 与 “技术民俗” 重组**
    
    - 概念定义：
        - “文化转场”（法国学者西班牙提出）：不同文化交集后的建构过程，需通过语境调查，追踪概念在跨语言 / 跨领域传播中的语义变化。
        - “技术民俗”：对传统手工艺的技艺流程、地方知识、师徒传承、民俗意义产品的综合研究。
    - 民俗学的技术研究局限：聚焦工匠故事 / 行业组织，用 “故事类型法” 替代技术流程研究，忽略材料、设计、流通等全链条，且不涉及 “技术输入输出”（如丝绸之路技术传播）。
    - 重组价值：通过 “文化转场” 视角，研究技术民俗的跨文化传播（如玄奘西行、一带一路中的手工艺交流），填补民俗学 “过程研究” 空白。
    - 衍生概念群：二元过程、主流话语、制造能力。



如何理解跨文化？

其核心是**突破单一文化（或国家、区域）的边界局限，以多元视角探索文化间的对话、互鉴与共性，最终服务于人类共同体的文化发展**。结合文章内容，可从以下四方面具体理解 “跨文化” 的内涵：

### 一、跨文化的核心立场：摒弃 “中心论”，立足 “人类共同发展”

董晓萍明确指出，“跨文化” 的首要前提是**超越文化本位主义与单一中心论**—— 这一立场直接针对民俗学发展中的关键瓶颈（如芬兰学派的 “欧洲一元论”，主张全世界故事起源于欧洲、非欧洲国家需遵循欧洲模式）。

在跨文化视角下：

1. **拒绝 “文化霸权”**：不将某一文化（如欧洲文化、本土文化）视为 “标准模板”，而是承认各文化的多样性与特异性（如中国民俗学对 “口头传统 + 历史文献” 的双重研究，区别于西方民俗学的 “口头传统单一路径”）；
2. **立足 “人类共同体”**：从民俗学的 “本土立场” 转向 “人类社会共同发展立场”，不再仅聚焦一国内的民俗研究（如中国民俗学服务国家建设的 “国家学” 属性），而是关注 “世界多边交流问题”（如全球疫情中不同文化对 “口罩” 的认知差异与互动）。

### 二、跨文化的研究逻辑：突破 “本土局限”，衔接 “多元与共性”

董晓萍认为，民俗学因 20 世纪 “一战”“二战” 的历史背景，长期承担 “本土国家解放、民族独立” 的使命，形成了 “侧重本土研究” 的属性（如中国民俗学的 “文化学” 定位：与多民族文化共建），但这也导致其难以解决 “分布式民俗类型与异文差异”“民俗史 - 家庭史 - 社会史脱节” 等问题。

而 “跨文化” 的核心逻辑，是**打破这种 “本土闭环”，在 “多元文化差异” 与 “人类文化共性” 间建立衔接**：

- 一方面，承认文化的 “异质性”：不回避不同文化的差异（如中西 “口罩观” 的巨大分歧，中方重视防护、西方初期排斥），而是将差异视为 “跨文化对话的起点”；
- 另一方面，挖掘文化的 “共享性”：从差异中提炼人类共通的文化要素（如 “故事类型”—— 全球重大危机（自然灾害、疫情）中，不同文化都会通过 “故事叙事” 构建协同机制，口罩故事最终成为 “公共卫生共同体” 的文化载体）。

### 三、跨文化的核心支撑：三大 “概念重组”，构建研究框架

董晓萍提出，“跨文化” 的实践需通过 “民俗学概念与跨文化概念的重组” 落地，这三大重组是理解 “跨文化如何运作” 的关键，也界定了其研究范畴：

#### 1. “跨国” 与 “类型” 重组：超越地理边界，提炼共享文化符号

- “跨国” 的内涵：不仅是 “跨越国家疆域”，更指**遴选人类多元文明中 “可共同认知、欣赏、传播” 的文化成分**（如故事类型、民俗符号），通过对话推广其价值（如疫情中 “口罩故事” 的跨文化传播：从初期的认知冲突，到后期通过援助、政策协调形成 “跨文化共识”，成为人类命运共同体的文化实践）；
- “类型” 的作用：以民俗学的 “故事类型”（如阿尔奈 - 汤普森的 2500 个编号故事）为基础，突破 “一国故事研究” 的局限，发现不同文化中 “类型的共性”（如各国都有 “灾难应对” 的故事叙事），同时尊重 “类型的异文差异”（如中西口罩故事的不同表达形式）。

这一重组的核心是：**让 “不可全球化的民俗特色”（如某一民族的独特仪式）与 “可跨文化的民俗要素”（如故事类型）分离，聚焦后者的共享价值**。

#### 2. “多极点” 与 “一国” 重组：衔接 “本土研究” 与 “人类共有民俗”

民俗学长期存在 “一国民俗研究” 与 “人类共有民俗研究” 的割裂（如忽视婚制、继承制、丧葬制的世界共性，或因忌惮 “欧洲一元起源论” 而回避 “人类共有民俗” 研究）。

“多极点” 概念的引入，正是为了**在 “本土文化” 与 “人类共性” 间建立理论衔接**：

- 对 “多元来源民俗” 国家（如历史上经历分裂、侵略的国家）：可将 “自我民俗” 与 “他者民俗” 共同纳入研究，探索文化交融的历史经验；
- 对 “连续历史来源民俗” 国家（如中国）：可通过 “文化圈研究”（如丝绸之路中的民俗互动），在 “本土民俗”（如中国农业文明的技术民俗）与 “外来民俗”（如佛教传入后的民俗变异）中发现人类共有规律（如 “技术传承中的师徒关系” 是跨文化共通的民俗逻辑）。

典型案例是 “母系社会研究”：董晓萍指出，以往学界因 “直线进化论” 排斥母系社会的多样性，但跨文化视角下，结合考古资料与多国民俗（如西班牙、波罗的海国家的新考古发现），可重新确认 “母系社会是人类多元社会形态的重要组成”—— 这正是 “多极点” 视角对 “单一历史叙事” 的突破。

#### 3. “文化转场” 与 “技术民俗” 重组：关注 “空间过程” 与 “语义变化”

董晓萍强调，民俗学的跨文化研究多为 “时间线性比较”（如中日跨文化故事比较，仅关注文本异同），而 “跨文化” 的核心是 **“空间过程研究”**—— 即追踪文化在跨语境传播中的 “语义变化与建构”，这一实践通过 “文化转场” 与 “技术民俗” 的重组实现：

- “文化转场”（法国学者米歇尔・西班牙提出）：指不同文化交集时，概念（或符号）在 “跨语言、跨领域” 传播中的语义重构（如西方 “科学观” 传入中国时，通过传教士的翻译、改写，与中国民间 “天文观测” 知识融合，形成新的技术民俗概念）；
- “技术民俗”：指传统手工艺的 “技艺流程、地方知识、师徒传承” 等综合体系（如中国农业文明中的手工铸铜、丝绸技艺）。

二者重组的价值在于：填补民俗学 “技术研究” 的空白 —— 以往民俗学仅关注 “工匠故事”（如能工巧匠的传说），而跨文化视角下，通过 “文化转场” 追踪 “技术民俗的跨文化流动”（如丝绸之路中中国丝绸技艺的外传、外来机械艺术的内传），既关注 “技术流程本身”，也关注 “技术背后的文化语义变化”（如中国 “天作人合” 的技艺理念对西方工艺观的影响）。

### 四、跨文化的终极目标：从 “文化研究” 到 “价值共建”

董晓萍指出，“跨文化” 的最终指向不是 “文化比较”，而是 **“人类优秀文化的共享与价值共建”**：

- 短期目标：解决民俗学的基础研究瓶颈（如构建 “民俗史 - 家庭史 - 社会史” 的理论模型，避免在人文学科群中 “失语”）；
- 长期目标：在全球化与网络信息化背景下，构建 “更高层次的人文精神”—— 通过跨文化研究，促进 “保护自我文化” 与 “尊重他者文化” 的平衡（如非遗保护不再是 “一国的应用研究”，而是 “人类共同遗产的历史性研究”），最终服务于 “人类命运共同体” 的文化建设（如口罩故事推动 “公共卫生共同体”，技术民俗流动推动 “文明互鉴共同体”）。

### 总结：董晓萍语境下 “跨文化” 的本质

简言之，董晓萍笔下的 “跨文化”，是**以 “人类共同体” 为价值核心，以 “突破边界、多元对话” 为方法，以 “解决民俗学瓶颈、提炼共享文化” 为路径的学科实践**—— 它不是 “某一文化的延伸”，也不是 “文化差异的消解”，而是在承认差异的基础上，构建 “各美其美、美美与共” 的文化研究体系，最终实现人文学科的整体整合与人类文化的共同发展。
