
一、研究背景与核心对象​

1. 研究困境：长期以来，口头文学生产者被视为 “面目模糊的集体”，理论讨论空间受限；晚近西方学界虽增多杰出演述人个案研究并进行理论概括，但仍有诸多可探讨问题。​

2. 核心聚焦：以口头文学演述人为研究对象，通过与书面文学作家的对比，从创作媒介、主体属性、运行策略、身心调用等维度，揭示演述人的特性，为建构 “非文字文学传统”（即口头诗学）理论体系奠基。​

3. 概念界定：​

- 演述人：所有类型口头文学的生产者与传承者。​

- 文学分类：按媒介划分为 “口头文学” 与 “书面文学”，替代传统按生产者划分的 “民间文学” 与 “作家文学”。​

二、创作媒介：“言” 与 “文” 的差异与关联​

1. 核心差异：​

- 口头文学：演述人用 “言”，传播与接受依赖 “口耳相传”。​

- 书面文学：作家用 “文”，接受依赖 “文本阅读”。​

2. 关联现象：言文互缘：​

- 口头文学受书写文化影响：演述人群体可能通过多种方式接触书写文化。​

- 书面文学借鉴口头传统：部分作家化用民间诗性智慧（如中国屈原、俄国普希金），或转化民间口头诗歌为书面读物。​

3. 关键原则：​

- 二者并非 “缠绕交叠可混合” 的信息技术，而是遵循不同法则：即便现代口头诗人掌握文字，仍以口头方式创编；文学生产多 “非‘言’即‘文’”，因二者适用于不同情景、承担不同社会文化功能，共同构成人类文学整体图景。​

三、生产属性：集体性创编与个人化写作的对立​

|   |   |   |
|---|---|---|
|对比维度​|口头文学（演述人）​|书面文学（作家）​|
|生产主体​|集体（创编主体、传承参与、内容归属均为集体）​|个人（私人空间内独立创作）​|
|内容归属​|群体共享（无专属演述人，属社区 / 民族等）​|作家专属（作品版权归个人）​|
|创作核心​|“创编”（因袭传统，需掌握 “曲目库”）​|“写作”（原创性为主，无固定 “曲目” 约束）​|
|杰出者能力​|增删情节、调用多文类、调动受众情绪​|构建完整文本、追求个性表达​|
|变异制约​|受受众 “期待视野”（前理解基础）制约，“在限度内变异”​|无实时受众制约，作品定稿后可修改但常终止创作​|

1. 口头文学的集体性：学界普遍认可 “集体性” 是口头文学的基本特征，具体体现为 “创编主体集体化、传承过程集体参与、叙事内容集体所有”。​

2. 演述人的个体能动性：​

- 杰出演述人需投入更长时间学习，掌握庞大 “曲目库”，能胜任史诗、叙事诗等大型文类演述。​

- 演述中可灵活调整：增删情节、细化场景、调用多文类，甚至与受众互动掀起演述高潮。​

四、运行策略：非中心化与中心化的传播格局​

1. 口头文学：非中心化（类比区块链）：​

- 区块链特性与口头文学的契合点（表 1）：​

|   |   |
|---|---|
|区块链特性​|口头文学对应表现​|
|去中心化​|无核心节点，群体成员均为 “信息存储节点”​|
|开放性​|内容公开，无独占 / 封锁，所有人可参与​|
|独立性​|不依赖第三方物质载体或制度保障​|
|匿名性​|演述人多不署名，文本无个人标识​|
|信息可验证​|内容通过多人口耳相传反复验证，维持稳定性​|

- 传播模式：“众多演述人对接众多受众”，演述与传播 “分布式进行”（在各处以多种方式发生）。​

2. 书面文学：中心化：​

- 传播模式：“一人（作家）对接万千读者”，作品依赖文字载体（如手稿、书籍），若载体毁损，作品可能消失。​

五、身心调用：具身性与离身性的分野​

1. 口头文学的 “具身性”：​

- 核心内涵：演述并非单纯 “语言表达”，而是演述人与受众 “身心共同参与” 的过程，体现在三方面：​

1. 演述过程：依赖演述人身体状态（表情、动作）与受众身体反馈（喜好 / 厌烦的身体语言），实时调整演述节奏、情节详略。​

2. 技艺习得：通过 “口传心授”，涉及身心多维度调动，非单纯 “知识传递”。​

3. 文本生成：因实时身心互动，同题故事易生成 “文本聚簇”（同题多异文），这是口头与书面文学的分水岭，也是口头文学 “新生性” 的根源。​

- 关键认知：口头文学的 “操演活动本身” 有时比 “唱词（文本）” 更重要，传统 “以作品为中心” 的书面文学研究视角在此失效。​

2. 书面文学的 “离身性”：​

- 核心内涵：创作无需实时身体参与，作家仅需将语言转化为书写符号，与作品呈 “分离” 状态。​

- 文本要求：追求 “完整性”，未完成作品常被其他作家 “续作” 以补全；作品定稿后，创作活动即终止，无实时变异可能。​

六、杰出演述人的价值重估​

1. 认知转折：​

- 传统认知：演述人被 “集体” 遮蔽，早期采录文本多无演述人信息。​

- 现代研究：20 世纪 30 年代起，帕里、洛德通过南斯拉夫史诗田野工作，首次系统关注演述人个体才艺，此后 “关注杰出演述人” 成为专业研究要求。​

2. 理论支撑：本雅明在 1936 年专论中指出 “讲故事是门手艺，故事中留有讲故事者的痕迹，就如陶坯表面留有陶匠的劳作痕迹那般”，辩证揭示了口头文学 “集体性” 与 “个体性” 的统一。​

3. 中国研究进展：近年中国学者关于杰出歌手的研究增多，虽未使用 “传奇歌手” 表述，但已肯定其对叙事传统的重要贡献及鲜明个人印痕，此类研究需引发学界更多关注。​

七、结论：口头诗学的核心观点与文学本质​

1. 口头与书面文学的终极差异（表 2）：​
**创作媒介**：演述人用 “言”，作家用 “文”；存在 “言文互缘”（如演述人受书写影响、作家化用民间智慧），但二者遵循不同信息技术法则，文学生产多 “非言即文”，适配不同场景与功能。
**生产属性**：口头文学是 “集体创编”，内容属群体共享，演述人需因袭传统、掌握 “曲目库”，且 “在限度内变异”（受受众 “期待视野” 制约）；书面文学是 “个人写作”，作品专属作家，创作具个人化属性。
**运行策略**：口头文学呈 “非中心化”，类似区块链的去中心化（群体 “分布式存储” 内容）、开放性（内容公开）、匿名性（演述人不署名）；书面文学呈 “中心化”，以作家为核心，依赖文字载体传播。
**身心调用**：口头文学演述具 “具身性”，需演述人与受众身心互动（影响演述节奏），生成 “文本聚簇”（同题多异文）；书面文学创作具 “离身性”，作品定稿后创作终止，追求文本完整性。

3. 文学本质认知：​

- 共性：二者均为 “媒介的艺术性使用”（语言或文字的艺术化表达）。​

- 核心属性：因书写本质是 “语言的记录”，故口头性与 “语言性” 是文学的核心属性；深入理解口头性，有助于更好把握语言的书写艺术。​

- 价值判断：文学高下不取决于 “口头” 或 “书面” 形式，而取决于生产者的 “精神境界” 与 “艺术造诣”—— 杰出作家多为见识卓著的思想家，伟大思想也常深藏于普通民间演述人心中。​

3. 文本命运：无论是口头故事还是书面小说，均为人类创造物，具 “自主性”，其价值与生命力源于人类的创造性实践。
